<h2 id="the-loop" class="before:content-['##_'] before:text-accent-2 before:mr-1">The loop</h2>

One idea fans out into many parallel agent sessions — then converges into a single dashboard you can watch from your phone. `hop` and `tu` wrap the loop: navigation in, token tracking out.

<Diagram light="/diagrams/loop-light.svg" dark="/diagrams/loop-dark.svg" alt="The toolkit loop: idea, fab-kit, wt, run-kit, hop, tu" />

**Inside the loop:** `idea` captures it · `fab-kit` turns it into a spec the AI can't fudge · `wt` spins up an isolated worktree per change so N agents work in parallel without conflicts · `run-kit` gives you one browser tab to watch them all (mobile-friendly via Tailscale).

**Wrapping the loop:** `hop` jumps you into any worktree (or across repos) in one keystroke · `tu` tracks what all those parallel agents are costing you in tokens · `shll` keeps everything installed and updated.

<h2 id="install" class="before:content-['##_'] before:text-accent-2 before:mr-1">Install</h2>

<div class="relative">
  <span class="absolute top-2 right-3 text-xs uppercase tracking-wider text-fg-faint select-none pointer-events-none z-10">bash</span>

```bash
$ brew install sahil87/tap/all   # installs every tool in the toolkit
$ shll shell-install             # one command to wire shell integration into your rc file
$ exec $SHELL                    # reload so the integration takes effect
```

</div>

Later, `shll update` upgrades every installed tool in one go. See the [shll README](https://github.com/sahil87/shll) for the full command surface.

<h2 id="tools" class="before:content-['##_'] before:text-accent-2 before:mr-1">Tools</h2>

<div class="not-content font-mono text-sm my-4">
  <ToolRow name="idea">Plain-Markdown backlog (<code class="text-fg">fab/backlog.md</code>) — worktree-aware, queryable from the CLI, feeds <code class="text-fg">/fab-new</code>.</ToolRow>
  <ToolRow name="hop">Fuzzy-nav, batch-git, and run-anything-inside-any-repo from one <code class="text-fg">hop.yaml</code> config.</ToolRow>
  <ToolRow name="fab-kit">7-stage pipeline that forces AI agents to plan before they code.</ToolRow>
  <ToolRow name="wt">Opinionated <code class="text-fg">git worktree</code> wrapper — one worktree per change, zero conflicts.</ToolRow>
  <ToolRow name="run-kit">Browser dashboard for tmux + Claude Code workspaces. Mobile-friendly via Tailscale.</ToolRow>
  <ToolRow name="tu">Token/cost tracker for Claude Code, Codex, OpenCode. Multi-machine sync, live watch.</ToolRow>
  <ToolRow name="shll" branch="└──">Meta-CLI — <code class="text-fg">shll install / update / shell-install</code> to wire and maintain the whole toolkit.</ToolRow>
</div>
